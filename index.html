Sabor de amor doces

Projeto freelancer desenvolvido do zero para uma cliente que vende doces caseiros. Utilizei tecnologias web modernas e soluções criativas para automatizar processos com custo zero.

O Desafio

A cliente precisava de um site para divulgar seus produtos, exibir cardápio atualizado automaticamente e facilitar o contato com os clientes, tudo sem custos mensais ou sistemas complexos.

A Solução

  • HTML, CSS, JS e Bootstrap para front-end responsivo
  • Google Sheets como banco de dados
  • Consumo da API do Sheets com JavaScript
  • AppSheet vinculado às planilhas para melhor gerenciamento
  • Deploy na Vercel com variáveis de ambiente a fim de proteger dados sensíveis
  • Configuração de domínio personalizado

O Resultado

O site ficou leve, responsivo e funcional, com total autonomia para a cliente atualizar seu cardápio via app Google Sheets. A experiência de desenvolver algo real do zero me trouxe um aprendizado enorme. O que eu aprendi.

Aprendizado

Esse projeto foi o primeiro site real que eu desenvolvi, com isso vieram desafios e novos aprendizados. Eu senti uma evolução significativa em minha confiança no desenvolvimento web, aprendi melhor sobre a linguagem Javascript e alguns de seus conceitos como Funções, eventos e manipulação do DOM.

Tive a oportunidade de elaborar soluções eficientes diante de alguns desafios, o maior destaque vai para um problema que surgiu: "De inicio não havia um banco de dados!" o deploy do projeto foi feito na Vercel a qual não suporta banco de dados MySql, logo havia um problema pois até então eu só conhecia a Vercel. Minha solução foi a seguinte: Usar uma planilha do Google Sheet e gerar um app que manipula a planilha, ao mesmo tempo o site obtinha os dados que estavam nessa planilha, logo, quando a planilha era alterada pelo app os dados do site também eram atualizados no deploy!

Tive também a oportunidade de desenvolver soft-skills por manter contato frequente com a cliente, oferecendo suporte ao esclarecer dúvidas, solicitando aprovações, fornecendo sugestões e atualizando sobre o status do desenvolvimento.
